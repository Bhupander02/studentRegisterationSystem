# studentRegisterationSystem

A registeration system for recording details of student, and having the feature's to delete of edit their details.

The section below has a tabular format and every new entry is added to the list
The browser local storage has been utilised here, so whenever the page is refreshed the data is not lost.
Once the edit button is clicked, the user data is displayed in the designted sections and when the data is uploaded the original data is now changed and displayed in the data section also no duplicate entry is allowed is made sure of in case when the edited data is saved.
